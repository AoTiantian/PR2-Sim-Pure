# Admittance / Compliance 快速验证（cartesian_controller_simulation）

本文记录一套最小命令序列，用于在 `cartesian_controller_simulation` 的仿真环境中验证 **柔顺/导纳效果**是否工作（末端在外力扰动下产生预期位移/让步）。

## 前置条件

- 已在一个终端启动仿真（含 RViz）：

```bash
export LC_NUMERIC="en_US.UTF-8"
ros2 launch cartesian_controller_simulation simulation.launch.py
```

- 另开一个终端（用于发控制命令/话题），并完成环境 source：

```bash
source /opt/ros/jazzy/setup.bash
source /workspace/cc_ws/install/setup.bash
```

## 1. 切换到柔顺控制器（Compliance）

激活 `cartesian_compliance_controller`，并关闭可能抢同一组接口的控制器：

```bash
ros2 control switch_controllers \
  --activate cartesian_compliance_controller \
  --deactivate cartesian_motion_controller joint_trajectory_controller
```

## 2. 发送末端平衡位姿（Equilibrium Pose）

本仿真 launch 将 `cartesian_compliance_controller/target_frame` remap 为 `/target_frame`，因此直接向 `/target_frame` 发布：

```bash
ros2 topic pub --once /target_frame geometry_msgs/msg/PoseStamped "{
  header: {frame_id: base_link},
  pose: {position: {x: 0.4, y: 0.0, z: 0.4}, orientation: {w: 1.0}}
}"
```

## 3. 注入外力扰动（FT Sensor Wrench）

本仿真 launch 将 `cartesian_compliance_controller/ft_sensor_wrench` remap 为 `/ft_sensor_wrench`。

向传感器系（示例为 `sensor_link`）发布外力扰动，观察 RViz 中末端是否出现预期柔顺位移：

```bash
ros2 topic pub --once /ft_sensor_wrench geometry_msgs/msg/WrenchStamped "{
  header: {frame_id: sensor_link},
  wrench: {force: {x: 20.0, y: 0.0, z: 0.0}, torque: {x: 0.0, y: 0.0, z: 0.0}}
}"
```

你可以修改 `force.{x,y,z}` 方向/幅值来验证响应是否符合预期。

## 4.（可选）查看控制器状态

```bash
ros2 topic echo /cartesian_compliance_controller/state -n 1
```
